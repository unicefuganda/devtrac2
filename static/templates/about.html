About DevtracGlobal

DevtracGlobal is the real time reporting on Public Services in Uganda. The single page web application aggregates data from various sources and present them in a meaningful way to the user. The MAP based application is user responsive and really easy to use.

Some of the data sources are:
  • Devtrac
  • UReport
  • Unicef etc.
The application also allows for filtering of data based on many parameters available from the sources.
